## Getting Started

### Prerequisites

Make sure you have the following installed:

* C++ compiler (like `g++` or `clang++`)
* CMake build tool

### Installation

Clone and build the project:

```bash
git clone https://github.com/Xaron11/cpp-todo.git
cd cpp-todo
mkdir build && cd build
cmake ..
cmake --build .
```

---

## Usage

Launch the app from your terminal:

```bash
./todo
```
